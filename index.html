Tentang Kalz Download Clone

Ini contoh situs pengunduh video berbasis web. Semua proses download sebenarnya dilakukan oleh server / API milik kamu sendiri (bukan dari GitHub Pages).

Fitur Utama

  • UI modern, responsif, enak di HP.
  • Tanpa login atau install aplikasi.
  • Deploy gratis di GitHub Pages.
  • Bisa dihubungkan ke beberapa server / API cadangan.
